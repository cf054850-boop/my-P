מסמך דרישות לפרויקט גמר

HTML / CSS / JavaScript - אתר או משחק מחשב

1. מטרת הפרויקט

הפרויקט נועד להציג שליטה מלאה בטכנולוגיות HTML, CSS ו-JavaScript באמצעות פיתוח אתר אינטראקטיבי או משחק מחשב בדפדפן. הפרויקט צריך להדגים יכולות תכנון, כתיבת קוד מסודר, עבודה עם DOM, ניהול נתונים, ואינטראקטיביות מלאה מול המשתמש.

  • מערכת ניהול משימות
  • ועוד

ב. משחק מחשב בדפדפן

  • משחק זמן תגובה
  • ועוד

3. דרישות חובה

3.1 מערכת משתמשים
