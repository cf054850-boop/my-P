מבוא

מבוא ל-DOM

ה-DOM (Document Object Model) הוא ייצוג של מסמך HTML כמבנה עץ של אובייקטים. כאשר הדפדפן טוען דף HTML, הוא מפרש את הקוד ובונה מבנה נתונים בזיכרון — עץ של צמתים (nodes) — שכל אחד מהם מייצג תגית, טקסט או תכונה.

JavaScript ניגש לעץ זה דרך אובייקט גלובלי בשם document, ומאפשר לקרוא, לשנות, להוסיף ולמחוק אלמנטים באופן דינמי, ללא טעינה מחדש של הדף.

// HTML: <p id="greeting">שלום עולם</p> // JavaScript — גישה לאלמנט: const el = document.getElementById("greeting"); console.log(el.innerText); // "שלום עולם"
השוואה ל-C#:
ב-C# (WinForms / WPF) גישה לרכיב UI נעשית דרך שם משתנה שמוגדר בקובץ ה-Designer:
this.lblGreeting.Text = "שלום";
ב-JS גישה לאלמנט נעשית דרך ה-DOM בזמן ריצה, לפי מזהה (id) או בורר CSS.

בחירת אלמנטים

בחירת אלמנטים ב-DOM — מבט כללי

על מנת לשנות תוכן, סגנון או התנהגות של אלמנט, יש תחילה לבחור אותו ולאחסן אותו במשתנה. JavaScript מציע מספר שיטות לכך.

  • בחירת אלמנט יחיד — מחזירה אובייקט אחד (או null)
  • בחירת קבוצת אלמנטים — מחזירה אוסף (NodeList / HTMLCollection)

בחירה יחידה

getElementById("id") querySelector(".cls")

בחירה מרובה

querySelectorAll(".cls") getElementsByClassName()
השוואה ל-C#:
ב-C# (WPF) ניתן למצוא אלמנט לפי שם:
FindName("myButton") as Button
ב-JS הבחירה מתבצעת בזמן ריצה לפי id, tag, class, או כל בורר CSS אחר.

בחירה יחידה

בחירת אלמנט בודד

יש שתי שיטות עיקריות לבחירת אלמנט בודד מה-DOM:

getElementById

// HTML: <div id="box">תוכן</div> const el = document.getElementById("box"); // מחזיר את האלמנט עם id="box", או null אם לא קיים

querySelector

// בחירה לפי id: const el1 = document.querySelector("#box"); // בחירה לפי class: const el2 = document.querySelector(".card"); // בחירה לפי תגית: const el3 = document.querySelector("h1");
  • getElementById — מהיר יותר, אך עובד רק עם id
  • querySelector — גמיש יותר, תומך בכל בורר CSS, מחזיר את האלמנט הראשון שנמצא
השוואה ל-C#:
ב-C# (WinForms) רכיב ניגש ישירות בשמו: button1.Text = "לחץ";
ב-JS יש לבצע בחירה דינמית לפני כל גישה לאלמנט.

מאפיינים

עבודה עם מאפיינים של אלמנטים

לאחר בחירת אלמנט, ניתן לקרוא ולשנות את תכונותיו המרכזיות:

const el = document.getElementById("title"); // קריאת טקסט: console.log(el.innerText); // שינוי טקסט: el.innerText = "כותרת חדשה"; // שינוי HTML פנימי: el.innerHTML = "<strong>מודגש</strong>"; // קבלת / שינוי ערך של input: const input = document.getElementById("nameInput"); const val = input.value; input.value = "ברירת מחדל";
הבדל חשוב — innerText לעומת innerHTML:
innerText מטפל בטקסט בלבד — HTML tags יוצגו כטקסט רגיל ולא יפורשו.
innerHTML מאפשר הכנסת תגיות HTML, אך דורש זהירות: קלט לא מאומת עלול לגרום לפגיעות XSS.
השוואה ל-C#:
ב-C# WinForms: label1.Text = "שלום";
ב-JS: el.innerText = "שלום"; — הגיון דומה, תחביר שונה.

סגנון

שינוי סגנון של אלמנטים

יש שתי גישות עיקריות לשינוי המראה החזותי של אלמנט:

שינוי ישיר דרך style

const box = document.getElementById("box"); box.style.color = "red"; box.style.backgroundColor = "#eee"; // camelCase! box.style.fontSize = "20px";
חשוב לשים לב: שמות מאפייני CSS שמכילים מקף (כגון background-color) נכתבים ב-JS בפורמט camelCase: backgroundColor.

הוספה והסרה של מחלקות CSS (classList)

box.classList.add("highlighted"); // הוספת class box.classList.remove("highlighted"); // הסרת class box.classList.toggle("active"); // החלפה: אם קיים — מסיר, אם לא — מוסיף box.classList.contains("active"); // true / false
השוואה ל-C#:
ב-WPF: btn.Style = (Style)FindResource("MyStyle");
ב-JS: עדיף להגדיר מחלקות CSS מראש ולהחליפן בעזרת classList, במקום לכתוב style inline.

אירועים

אירועים (Events) — מבוא

אירוע (Event) הוא פעולה שמתרחשת בדפדפן — לחיצה על כפתור, הקלדה, גלילה, טעינת דף ועוד. JavaScript מאפשר להגיב לאירועים אלו על ידי חיבור פונקציית handler לאלמנט.

  • click — לחיצה על אלמנט
  • input — שינוי ערך בשדה קלט
  • keydown / keyup — לחיצה על מקש
  • submit — שליחת טופס
  • mouseover / mouseout — מעבר עכבר
// מבנה כללי: element.addEventListener("eventName", function() { // קוד שיבוצע כשהאירוע מתרחש });
השוואה ל-C#:
ב-C# WinForms: button1.Click += new EventHandler(button1_Click);
ב-JS: btn.addEventListener("click", myFunction);
הרעיון זהה — חיבור פונקציה לאירוע. התחביר שונה.

אירוע click

אירוע click — בפירוט

האירוע הנפוץ ביותר ב-DOM הוא click. ניתן לחבר אליו פונקציה בשתי דרכים:

דרך 1 — addEventListener

const btn = document.getElementById("myBtn"); btn.addEventListener("click", function() { alert("הכפתור נלחץ!"); });

דרך 2 — Arrow Function

btn.addEventListener("click", () => { document.getElementById("msg").innerText = "נלחץ!"; });

שימוש בפונקציה מוגדרת מראש

function handleClick() { console.log("לחיצה זוהתה"); } btn.addEventListener("click", handleClick); // שים לב: מועברת הפנייה לפונקציה — ללא סוגריים!
השוואה ל-C#:
ב-C#: button1.Click += HandleClick; — גם כאן מועברת הפנייה ללא סוגריים.
ב-JS: btn.addEventListener("click", handleClick); — דפוס זהה.

קלט מהמשתמש

עבודה עם value בקלטים

שדות קלט (input) הם חלק מרכזי בכל טופס. בעזרת value ניתן לקרוא את מה שהמשתמש הזין, ולשלב אותו בלוגיקת הדף.

// HTML: // <input id="nameInput" type="text" placeholder="הזן שם" /> // <button id="submitBtn">שלח</button> // <p id="output"></p> // JavaScript: const btn = document.getElementById("submitBtn"); btn.addEventListener("click", () => { const input = document.getElementById("nameInput"); const name = input.value; if (name === "") { alert("יש למלא את השדה"); return; } document.getElementById("output").innerText = "שלום, " + name + "!"; });
  • ניתן להציג את הערך בדף (innerText), בקונסול (console.log), או ב-alert
  • חשוב לאמת את הקלט לפני שימוש (בדיקת ריקנות, פורמט וכד')
השוואה ל-C#:
ב-WinForms: string name = textBox1.Text;
ב-JS: const name = input.value; — מושג זהה, תחביר שונה.

דוגמאות משולבות

דוגמאות משולבות

להלן שילוב של כל המרכיבים: בחירת אלמנט, אירוע, קלט ועדכון ה-DOM.

דוגמה 1 — שינוי צבע ב-click

const box = document.getElementById("colorBox"); box.addEventListener("click", () => { box.classList.toggle("active"); box.innerText = box.classList.contains("active") ? "פעיל" : "לא פעיל"; });

דוגמה 2 — מחשבון פשוט

document.getElementById("calcBtn").addEventListener("click", () => { const a = Number(document.getElementById("num1").value); const b = Number(document.getElementById("num2").value); const result = a + b; document.getElementById("result").innerText = "סכום: " + result; });

דוגמה 3 — סינון רשימה

document.getElementById("search").addEventListener("input", () => { const term = document.getElementById("search").value.toLowerCase(); const items = document.querySelectorAll(".item"); items.forEach(item => { const match = item.innerText.toLowerCase().includes(term); item.style.display = match ? "block" : "none"; }); });
עיקרון מנחה:
כל פעולה ב-DOM עוקבת אחרי אותו תבנית: בחר אלמנט → האזן לאירוע → שנה את ה-DOM. תבנית זו חוזרת על עצמה בכל פרויקט JavaScript.
